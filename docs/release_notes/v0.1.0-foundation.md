# Release Notes — v0.1.0-foundation

**Tag**: `v0.1.0-foundation`
**Date**: 2026-07-05

Personal R&D Agent OS — first portfolio-ready release. All 17 batches shipped. Release gate green. Real local model runtime + real research corpus ingestion + LangGraph runtime + tool permission gate + 3 real research apps.

## Phase 1 (Batch 0–10) — Foundation

- ✅ uv + pyproject + rdos CLI skeleton
- ✅ Pydantic schemas + YAML config loader
- ✅ Heading-aware Markdown chunker with chunk_hash dedup
- ✅ SQLite metadata + LanceDB vector store (idempotent index)
- ✅ Hybrid retriever (semantic + keyword via RRF)
- ✅ Citation builder + three-way validator (chunk_exists ∧ hash_matches ∧ in_retrieved_context)
- ✅ Privacy Router (effective = max across query/chunks/tools/memory/trace)
- ✅ Model Router (data-only decisions, never binds tools)
- ✅ Local llama.cpp adapter + structured output formatter (Pydantic + retry once)
- ✅ 11-node research_memory_graph + `rdos ask`
- ✅ JSONL trace store + `rdos trace list/show`
- ✅ 8-metric release gate (all PASS)
- ✅ Portfolio polish (README, case studies, samples)

## Phase 2 (Batch 11–17) — Production realism

- ✅ **Batch 11** — `OpenAICompatibleEmbeddingProvider` for bge-m3 (1024-d batch); LLM mode `stub|local|auto`; provider/dim mismatch guards; `rdos doctor models`
- ✅ **Batch 12** — Real clawd-research ingestion (2,088 files, 25 topics); incremental index with content_hash dedup + stale markers; index reports; corpus presets (`--scope rag|agent|eval|security|devtools|all`)
- ✅ **Batch 13** — Query rewriter (English term preservation + CJK n-grams + alias expansion); `rdos benchmark retrieval` (Recall@3/5, MRR, hit rates, no-answer accuracy, latency p50/p95); 63-sample real eval set; no-answer handling
- ✅ **Batch 14** — LangGraph StateGraph runtime with InMemorySaver checkpointer; per-invoke thread_id; node-level trace; `--graph-runtime langgraph|linear`
- ✅ **Batch 15** — ToolRegistry + PermissionGate + CapabilityBoundary; safe tools (search_notes, read_note, list_recent_notes, export_report); path traversal / symlink escape / secret-name / max_bytes guards; `rdos tool` CLI
- ✅ **Batch 16** — Real research apps: `rdos research digest`, `rdos research topic`, `rdos research synthesize` with citation coverage metric
- ✅ **Batch 17** — Adversarial eval sets (50+ privacy routing, 50+ model routing, 40+ citation); benchmark report; demo scripts; portfolio README
