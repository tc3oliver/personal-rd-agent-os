## Known limitations (highlights)

Full list in [docs/limitations.md](../limitations.md).

- **No HITL approval UI.** `requires_approval` tools return `approval_required` but cannot complete.
- **No cloud escalation.** `private_summary` cannot safely escalate; falls back to local.
- **No multi-turn conversation.** Each `ask` is independent.
- **No-answer framework exists but is disabled by default.**
- **InMemorySaver only.**
- **Only 3 of 25 clawd-research scopes are benchmarked.**
- **v0.1.0 is not production-grade.**
