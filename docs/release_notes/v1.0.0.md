# Release Notes — v1.0.0

**Tag**: `v1.0.0`
**Date**: 2026-07-05
**Positioning**: **v1.0 is feature complete and frozen.** Maintenance-only policy after this tag.

Personal R&D Agent OS — first feature-complete release. Foundation layer (Phase 1), trust runtime (Phase 2 + 3), audit-driven hardening, and v1.0 freeze. The end-to-end loop (Markdown → index → retrieve → cite → privacy route → model route → generate → structure → validate → trace → eval → apps) is real, tested, and exercised on the real clawd-research corpus.

For full quality numbers see [docs/quality_baseline_v0.1.0.md](../quality_baseline_v0.1.0.md). For what's out of scope, see [docs/parking_lot.md](../parking_lot.md). For audit history, see [docs/audit/full_project_audit.md](../audit/full_project_audit.md).

## Limitations

See [docs/limitations.md](../limitations.md) for the full categorized list. Highlights:

- **No cloud adapter** — `cloud_send()` + recognizers ready; no sender shipped.
- **No LLM-as-judge** — answer quality is not auto-graded.
- **No-answer threshold defaults to 0** — calibrate before enabling.
- **InMemorySaver default** for LangGraph checkpointer (SQLite factory exists).
- **3 of 25 corpus scopes benchmarked**.
- **Not production-grade** — portfolio / foundation release.

## Non-goals (intentionally deferred — see [docs/parking_lot.md](../parking_lot.md))

- Cloud adapter (OpenAI / Anthropic / Gemini) — local-first is the design
- LLM-as-judge for answer quality
- Web UI / mobile app
- Plugin external sources (GitHub issues, arXiv, Hacker News)
- Multi-tenant / production multi-user deployment
- Knowledge graph rewrite
- Auto code editing
- `run_shell` / `git_push` / `send_email` / `delete_file` tools (out of scope forever)
