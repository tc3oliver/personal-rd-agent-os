## Blocking bug found and fixed

**Bug**: `_synthesize()` in `src/rdos/graph/export_graph.py:42-49` was setting `synthesis_md` to the file **path** (returned by `run_synthesis`) instead of the file **content**. The written report was 52 bytes containing only the path string.

**Fix**: One-line change — read the file content into `synthesis_md` before returning from `_synthesize`.

**Impact**: `export_graph.py` only (not used by `rdos research synthesize` CLI which writes directly). Bug was latent since Batch 19.

```diff
-    return {"synthesis_md": md, "synthesis_path": md}
+    md_content = Path(md_path).read_text(encoding="utf-8") if Path(md_path).exists() else ""
+    return {"synthesis_md": md_content, "synthesis_path": md_path}
```
